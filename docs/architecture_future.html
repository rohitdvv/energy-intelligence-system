Now vs Enterprise Scale

Current (Hackathon)

  • Streamlit monolith — single Python process
  • Parquet files on disk — single-user cache
  • Prophet only — one forecasting model
  • 3 agents — Bull / Bear / PM
  • State-level EIA data (Texas proxy issue)
  • No authentication — public app
  • fpdf2 for PDF — no Excel output
  • Streamlit Cloud deployment
  • Manual data refresh
  • Chat history lost on refresh

Enterprise Scale

  • FastAPI + React — decoupled, scalable
  • Snowflake + Redis — multi-tenant, concurrent
  • Prophet + XGBoost residual + TFT ensemble
  • 5 specialist agents with orchestrator
  • True basin-level data (RRC, Enverus)
  • JWT auth, RBAC, per-tenant isolation
  • PDF + Excel add-in + mobile push alerts
  • Kubernetes auto-scaling on AWS/GCP
  • Airflow DAGs — daily/weekly auto-refresh
  • PostgreSQL persistent chat history
